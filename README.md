# 🚀 Web3 AutoPost

Plataforma todo-en-uno para publicar contenido Web3 en todas las redes sociales automáticamente.

## 🎯 ¿Qué hace?

- Subes una imagen + texto + hashtags desde un solo panel
- Se publica automáticamente en **8 redes sociales** sin pedir aprobación
- Blog Web3 propio desplegado en Vercel
- Sin costo mensual

## 🌐 Redes sociales incluidas

### Para principiantes (Web2)
- Instagram
- TikTok
- X (Twitter)
- LinkedIn

### Para expertos Web3
- Farcaster (Warpcast)
- Mirror.xyz
- Lens Protocol
- Paragraph.xyz

## 🛠️ Stack tecnológico

- **Frontend**: Next.js 14 + Tailwind CSS
- **Blog**: MDX (Markdown + JSX)
- **Deploy**: Vercel (gratis)
- **Repositorio**: GitHub
- **Automatización**: n8n / Zapier
- **Panel de contenido**: Notion

## 🚀 Cómo usar

1. Clona este repositorio
2. Instala dependencias: `npm install`
3. Copia `.env.example` a `.env.local` y completa las variables
4. Corre el proyecto: `npm run dev`
5. Abre `http://localhost:3000`

## 📦 Variables de entorno

Ver archivo `.env.example` para todas las variables necesarias.
